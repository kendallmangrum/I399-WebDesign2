4x4 Magic Squares in Java

For one of my Computer Science homeworks I was assigned to create a program in Java that took in 16 numbers from the user and then tells them if the numbers they entered in that specific order created a 4x4 magic square. A magic square is a square of an equal number of rows and columns that are filled with numbers with each row, column, and diagonal adding up to the same number The image above shows a magic square. Here is the code for the program on GitHub.

Create the 4x4 Square

I decided to create the 4x4 square in the main of the program using an array of arrays. The program continually asks the user to input a number until they have entered 16 numbers, which fills up the 4x4 grid. The program checks how many numbers have been input after each new number so that it knows when to move on to the next row. The numbers fill in the grid starting at the top left and ending at the bottom right.

Getting the Sums

In order to be able to compare the sums of each row, column, and diagonal we have to add each one ourselves. To do so, I created an array that stores each sum of each row, an array that stores each sum of each column, and an array that stores the sums of each diagonal. I use a for loop to get the sums for each. I also check to make sure that only numbers 1-16 are used.

Comparing the Sums

After finding all of the sums, I compare them all to make sure that they are all the same. If they are all the same and only numbers 1-16 are used, the user did in fact input a Magic Square. If either one of those conditions aren't met, the user did not inout a Magic Square.

Visually Showing the Array

In order to visually show the magic square that the user input, I created a method called show. The method aligns the input numbers in 4x4 grid.
