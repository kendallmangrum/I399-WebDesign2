Snake Game Created in Java

For one of my Computer Science homeworks I was assigned to complete a custom game of Snake using what I had learned in Java. The game was supposed function similarly to the original game. Some of the code was provided as a starting point, but it was up to me to get the game to work as it is supposed to. Here is the code for the program on GitHub.

Draw the Snake

One of the most important parts of the program is actually drawing the snake so that it is visible for the user. The code goes through each piece of the snake starting with the head and draws each circle. This section of the code also shows how the snake is expanded, mkaing use of the direction of the snake. In order to make the program faster, the snake is expanded at the head and reduced at the end allowing it to update faster.

Generate Food to Grow

An important part of the game of Snake is the circles of food that cause the snake to grow in length. The program generates five pieces of food that are placed randomly within the game screen. The food is colored yellow so that it sticks out and is easy to identify compared to the snake which has a red head and blue body.

Remove the Food

The code that removes the food also helps to call the method that grows the snake. The food circle is removed from the screen and added internally to a list that used to know how much longer to make the snake. The image also shows how when food is removed, an equal amount is also added onto the screen in new, random positions.

Collision

The most integral function of the Snake game is the collision. Whenever the snake collides with a piece of food, the food is removed from the screen and another body segment is added to snake. Ultimately, the snake is going to either collide with itself or the edge of the screen. When this happends, the game is over.
